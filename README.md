# KeilaCred - Site Institucional para Financeira

## Sobre o projeto

O KeilaCred é um projeto frontend desenvolvido como atividade prática para simular o site institucional de uma empresa financeira local.

O objetivo foi criar uma aplicação web composta por múltiplas páginas, apresentando os principais serviços oferecidos pela empresa, além de áreas de simulação de empréstimos e cadastro de clientes.

Este projeto foi desenvolvido com foco na prática de HTML5 e CSS3, reforçando conceitos de estruturação de páginas, formulários, navegação e estilização de interfaces.

## Funcionalidades

### Página Inicial

- Área de destaque para ofertas (simulação de carrossel de imagens)
- Apresentação dos principais serviços
- Navegação entre páginas
- Rodapé com redes sociais e contatos

### Página Empréstimos

- Exibição dos tipos de empréstimo disponíveis
- Simulação de empréstimo
- Seleção de valor desejado
- Seleção da quantidade de parcelas
- Área de exibição do valor da parcela
- Seção de Perguntas Frequentes (FAQ)

### Página Seja Nosso Cliente

- Área institucional com imagem e texto descritivo
- Formulário de cadastro
- Seleção do serviço desejado
- Opção para informar se é aposentado

## Tecnologias utilizadas

- HTML5
- CSS3


2. Abra o arquivo `index.html` no navegador.
